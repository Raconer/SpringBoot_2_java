# Spring Boot 2.x
> Spring Boot 사용법 및 정의<br>
> master Branch 뿐 아니라 기능 및 특정 목적용 Branch에 따라 소스 코드 구조가 전부 다르니 필요한 내용의 Branch 를 확인 필요
## Branch 구조
* master
  * Core 개발 Master Branch 
* feature/***
  * Core 기능 개발 용 Branch
* \*\*\*
  * Mybatis, JPA 등 특정 목적의 Branch
* \*\*\*/feature/\*\*\*
  * Mybatis, JPA 등 특적 목적 Branch의 기능 개발용 Branch
